# Aula 02 — CSS básico (Site do Batman)

## Objetivo da aula

Aplicar CSS básico para melhorar a organização visual de um site: seletores (tag, `.class`, `#id`), cores, tipografia, box model e um menu simples com `display: flex`.

## Capacidades (SENAI)

- CT7: Aplicar propriedades e seletores CSS de forma correta.
- CS3: Organizar arquivos e manter padrão de código.

## Conteúdos-chave

- CSS no `<style>` (introdução)
- Seletores: `*`, tag, `.class`, `#id`, `footer p`
- Box model: `margin`, `padding`, `border`, `border-radius`
- `box-sizing: border-box`
- `display: flex`, `gap`
- Pseudo-classe `:hover`

## Arquivos desta aula

- `index.html` — “Site do Batman” com CSS interno
- Imagens: `batman_1.webp`, `batman_2.jpg`, `batman_3.jpg` (ou `assets/` se você mover)
