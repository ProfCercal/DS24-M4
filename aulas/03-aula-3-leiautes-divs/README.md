# Aula 03 — Website com leiaute por divs (Site do Palmeiras)

## Objetivo da aula

Desenvolver uma página temática sobre a Sociedade Esportiva Palmeiras utilizando **HTML e CSS**, com foco na construção de **leiautes por meio de `div`**, organização visual do conteúdo e aplicação de estilos exclusivamente em CSS externo.

## Tema do projeto

O website apresenta informações históricas, visuais e esportivas do Palmeiras, organizadas em diferentes seções:

- apresentação histórica do clube;
- escudos e momentos marcantes;
- elenco atual em um campo estilizado;
- camisas da temporada;
- tabela de títulos.

## Conteúdos trabalhados

Nesta aula, o foco foi:

- uso de `div` como contêiner de leiaute;
- diferença entre `div` e tags semânticas;
- organização de blocos visuais em HTML;
- estilização com `class` e `id`;
- uso de `display: flex` e `display: grid`;
- uso de `position` para distribuir elementos no campo;
- aplicação de efeitos visuais com `hover`, `scale`, `translate` e transições;
- organização de imagens e conteúdos em seções.
