# DS24 M3 (Turma) — Materiais, Aulas e Atividades

Este repositório é a pasta central da turma **DS24 M4**. Aqui ficam os exemplos feitos em sala, roteiros e bases para atividades.

## Como usar (alunos)

1. Encontre a aula/atividade no índice abaixo.
2. Abra a pasta correspondente e leia o `README.md` daquela pasta (lá está o passo a passo).
3. Quando for atividade com entrega, siga as instruções de fork/clone informadas no README da atividade.

## Organização do repositório

- `aulas/` → exemplos e códigos guiados (feitos em aula)
- `atividades/` → bases e enunciados das atividades (com critérios/rubricas)
- `assets/` → arquivos compartilhados (se houver)
- `templates/` → modelos de README (para replicar nas próximas aulas)

## Padrões gerais (para atividades)

- Arquivos: `index.html` e `style.css` (salvo orientação diferente).
- Organização: imagens em `assets/`.
- Boas práticas: classes para estilos repetidos; `id` para âncoras/elementos únicos.
- Git: commits por etapa (evite commit único no final).

## Regras de entrega (quando aplicável)

- A entrega será o link do repositório do aluno (fork ou repositório próprio), conforme orientado no README da atividade.
- Código deve abrir no navegador sem depender de servidor, salvo aviso.
